Configuration

cargo-hakari is configured through Hakari.toml at the root of the workspace.

Example configuration:

resolver = "2"
hakari-package = "workspace-hack"

platforms = [
    { triple = "x86_64-unknown-linux-gnu", target-features = "unknown" }
]
