pub fn dup_stderr() -> Result<PipeWriter>
Expand description

Get a duplicated copy of the current process’s standard error, as a PipeWriter.

Writing directly to this pipe isn’t recommended, because it’s not synchronized with std::io::stderr. PipeWriter implements Into<Stdio>, so it can be passed directly to Command::stdout or Command::stderr. This can be useful if you want the child’s stdout to go to the parent’s stderr.
